Добавление новой встречи:
1) Добавить новую встречу:
	- Выбрать пункт меню *Добавить встречу*
	- Ввести название для встречи
	- Ввести дату и время начала встречи в формате dd.MM.yyyy HH:mm где dd - день, MM - месяц, yyyy - год, HH - час, mm - минута
	- Ввести дату и время окончания встречи в таком же формате
	- Ввести время в минутах, за сколько до начала встречи необходимо сделать уведомление
	- Проверить, добавилась ли встреча, выбрав пункт меню *Показать список встреч*
	- Для необходимости повторить столько раз сколько нужно

2) Редактирование встреч:
	- Выбрать пункт меню *Редактировать*
	- Появится список добавленных встреч, из этого списка выбрать номер встречи(он указывается в начале строки), которую хотите редактировать
	- Ввести новое название для встречи
	- Ввести новую дату и время начала встречи в формате dd.MM.yyyy HH:mm где dd - день, MM - месяц, yyyy - год, HH - час, mm - минута
	- Ввести новую дату и время окончания встречи в таком же формате
	- Ввести время в минутах, за сколько до начала встречи необходимо сделать уведомление
	- Проверить, обновилась ли встреча, выбрав пункт меню *Показать список встреч*
	- Для необходимости повторить столько раз сколько нужно

3) Информирование о предстоящей встрече:
	- Добавить встречу
	- Ждать уведомления в консоли. Уведомление появится за столько минут до начала встречи, сколько было указано при создании

4) Пересечение встреч:
	- Создать встречу
	- Попробовать создать встречу, которая по времени будет пересекаться с уже созданной
	- В консоли должно появится предупреждение о пересечении с другой встречей. Создание будет прервано
	- Создать вторую встрчу которая не будет пересекаться с уже созданной
	- Изменить время, чтобы она пересекалась с уже созданной, использую редактор встреч
	- В консоли должно появится предупреждение о пересечении с ранее созданной стречей. Редактирование будет прервано

5) Сохранение в текстовый файл
	- Создать встречи с разной датой
	- Выбрать пункт меню *Сохранить в текстовый файл* 
	- Ввести дату в формате dd.MM.yyyy где dd - день, MM - месяц, yyyy - год
	- В Консоли отобразится путь сохраненного файла
6) Завершить работу
	- Выбрать пунк меню *Завершение работы*
	- Приложение должно перестать работать
